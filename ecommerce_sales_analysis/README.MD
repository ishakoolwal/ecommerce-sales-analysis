# E-Commerce Sales Data Analysis

This project analyzes an E-commerce dataset using Python to uncover
key sales insights.

Tools Used:
• Python
• Pandas
• Matplotlib
• Seaborn

Key Analysis:
• Sales by Category
• Regional Sales Performance
• Monthly Sales Trend
• Top Selling Products
• Customer Segment Analysis
• Top Cities and States by Sales

The analysis helps identify important sales patterns and high-performing
products and markets.
